Privacy

This application does not intentionally collect, store, sell, or profile personal data from users within the game itself.

No in-app data collection

The game runs locally in your browser and does not include user accounts, analytics, advertising trackers, telemetry dashboards, or custom backend services. Inputs such as launch settings and maneuvers remain in your current browser session.

GitHub-hosted delivery

If this site is served through GitHub Pages, GitHub may process technical request data required to deliver and secure the site. That can include standard web-hosting information such as IP address, request metadata, and server logs under GitHub's own infrastructure and policies.

For details on GitHub's data handling, please refer to the GitHub Privacy Statement: GitHub Privacy Statement.

Third-party services

Aside from the GitHub hosting layer, no additional third-party data collection is intentionally enabled by this project.
