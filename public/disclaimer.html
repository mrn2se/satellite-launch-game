Satellite Launch Game

Disclaimer

This project is a vibe-coded educational tool. It is intended to help visualize orbital mechanics and gravitational interactions, not to provide authoritative scientific, engineering, or mission-planning results.

No guarantee of correctness

No guarantee is given that the simulation, derived trajectories, visualizations, constants, scales, or gameplay outputs are correct, complete, or suitable for real-world use. The project deliberately simplifies parts of celestial mechanics and exaggerates some visuals to make the simulation playable and easier to understand.

If you need reliable scientific values or operational guidance, use verified primary references and specialist tools instead of this game.

Used sources

  • NASA JPL Horizons and related NASA reference material for orbital and planetary reference data
  • IAU reference data and naming conventions for bodies in the solar system
  • CODATA physical constants for gravitational and related baseline values
  • Three.js documentation for rendering APIs and WebGL integration patterns

Modeling notes

The application uses a simplified educational model. Planet sizes are visually exaggerated, control inputs are gamified, and the simulation is optimized for browser performance and learning value rather than scientific fidelity.
